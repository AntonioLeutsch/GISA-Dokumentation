Wordpress

WordPress wurde als freie Webanwendung zum Aufbau und zur Pflege von Weblogs konzipiert Dank einer großen Nachfrage und einer stetig wachsenden Community wurde aus WordPress eins der meist genutzten Contest-Management-Systeme weltweit.

Durch die besondere Wertsetzung auf Webstandards, Benutzerfreundlichkeit und gute Anpassbarkeit, kann mit Hilfe des Kommentar Systems, der Benutzerrollen Verwaltung und der zahlreichen Plug-ins ein besonders hochwertiges und stabiles System gewährleistet werden.

Aus diesen und weiteren Gründen, hat die GISA zusammen mit ./nviimedia+ entschieden WordPress als System für die neue Web Präsenz zu wählen.

GISA Theme

Auf der Basis der Designentwürfe, und mit der von WordPress vorgesehenen Struktur in Verbindung mit HTML 5, CSS 3 und Javascript 6, wurde ein Theme entwickelt welches und sämtliche Grundstruktur von WordPress abdeckt. Um schnelle Updates der Hauptfunktionen bei Versions- Sprüngen von WordPress schnell und sauber abzuarbeiten, wurde für die individuellen Funktionen des GISA Systems ein Child Theme erstellt, welches das Parent Theme um die individuellen Funktionen erweitert.

GISA Options
Die GISA Options sind die Backend Einstellmöglichkeiten zum Theme, sie wurden zur einfachen Umstrukturierung des gesamten Themes angelegt. Sie dienen auch zur Visuellen darstellung der im Main Theme verankerten Funktionen und Möglichkeiten.

Slider & Carousel

GISA Slider
Der Gisa Slider hat diverse Funktion von Haus aus mitbekommen, die Hauptfunktion für das Projekt ist die Darstellung eines großen Hero Images in Verbindung mit einer Headline. Alternativ gibt es die Möglichkeit ein Video in den Hintergrund zu laden. Die Slider werden über eine eindeutige Kategorie und dessen Shortcode auf die gewünschten Seiten geladen.

GISA Carousel
Über das Gisa Carousel können Logos von Partnerschaften, Zertifizierungen etc. eingeplegt und anschließend als Visual Composer Element auf Seiten eingebunden werden.

Custom Post Types

Custom Post Types
Die Custom Post Types beinhalten Inhaltselemente die nicht von Wordpress vorgegeben werden, aber der Wordpress Struktur und Funktion folgen. Für eine Bessere Verwaltung wurden für diverse Inhalte z.B. "Referenzen" Custom Post Types angelegt, und mit einer eigenen Inhaltsstruktur versehen. Dadurch können im Visual Composer diverse Elemente erzeugt werden, diese beziehen ihren Conten aus den Post Types.

Cool Timeline

Timeline Stories
Über die Timeline Stories können Beiträge erstellt werden, welche im Frontend der Seite in einem Zeitstrahl dargestellt werden. Dabei wird die Timeline nach dem Story Year sortiert, optional gibt es die Möglichkeit ein Bild für die Story zu hinterlegen.

Formulare

Formulare
Über die Formulare werden diverse Funktionen zur Verfügung gestellt, mit der Möglichkeit "Hooks" und Erweiterungen zu ergänzen gibt es hier nahezu unendlich Möglichkeiten. Die Hauptfunktion ist dabei klar definiert, die Möglichkeit schnell und einfach ein Kontaktformular mit der dazugehörigen E-Mail Template Struktur zu erstellen. Als Erweiterung wurde die Möglichekeit hinzugefügt ein Formular an ein Download Bereich zu binden.

Downloads

Downloads
Über die Downloads können diverse Datei Typen zum Download angeboten werden, dabei ist auch die Möglichkeit geschaffen ein individuelles Icon festzulegen. In Verbindung mit den Formularen wurde eine Erweriterung hinzugefügt welche den Download erst nach absenden eines Formulars ermöglicht.

Visual Composer

Der Visual Composer ist die beste Lösung für ein komplexes Frontend welches jeder Benutzer einfach und unkompliziert aufbauen kann. Er wandelt Shortcodes mit Inhalten in Grafische Elemente um. Für die Gisa wurde dieser erweitert um die eigenen Funktionen ohne weiteren Aufwand darzustellen.

M.E. Calendar

Der Modern Events Calendar bringt eine solide Basis Events zu erstellen und eine Buchungsoption zu jedem Event beizufügen. Für die Zahlungsmodule wurde das Plugin erweitert, damit jede Zahlungsmethode der Gisa vorhanden ist.

Mailpoet

Eigentlich ist Mailpoet ein komplettes Newsletter System. Für das Projekt wird allerdings nur die Anmeldefunktion benötigt, die gesammelten Daten lassen sich dann einfach im Dashboard exportieren. Das Plugin sammelte von Sich aus die IP-Adressen der Abonennten. Diese werden vom Theme aber mit dem Null Wert 127.0.0.1 überschrieben.

{SNAP} Social Network Auto-Poster

Das Plugin erlaubt es direkt aus Wordpress Beiträge zu diversen Social Media Kanälen zu posten. Dabei lässt sich über die Einstellungen die Ausgabe auf jedem Kanal individuell einstellen.

SEO & Performance

YOAST SEO
YOAST SEO, ist die All-in-one Lösung im SEO Bereich. Nahezu alle SEO relevanten Einstellungen und Funktionen können hier festgelegt werden.
W3 Total Cache
Das effizienteste Plugin für die Verwaltung der Cache Funktionen. Dabei ist bei dem Einstellen immer ein Auge auf das Front End zu werfen, hier können kleine Änderungen das Front End unerreichbar machen. Mit den richtigen Einstellungen kann hier ein massiver Performance Boost erreicht werden.
Reset IP Adresse
Ein Helfer Plugin von nviimedia für die anonymisierung der IP Adresse der Nutzer.
WP Better Email
Stellt Emails aus Wordpress mit dem HTML Content Type dar.
Duplicate Post
Ermöglicht das duplizieren von vorhanden Beiträgen und Seiten.
