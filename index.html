Wordpress

WordPress wurde als freie Webanwendung zum Aufbau und zur Pflege von Weblogs konzipiert Dank einer großen Nachfrage und einer stetig wachsenden Community wurde aus WordPress eins der meist genutzten Contest-Management-Systeme weltweit.

Durch die besondere Wertsetzung auf Webstandards, Benutzerfreundlichkeit und gute Anpassbarkeit, kann mit Hilfe des Kommentar Systems, der Benutzerrollen Verwaltung und der zahlreichen Plug-ins ein besonders hochwertiges und stabiles System gewährleistet werden.

Aus diesen und weiteren Gründen, hat die GISA zusammen mit ./nviimedia+ entschieden WordPress als System für die neue Web Präsenz zu wählen.

GISA Theme

Auf der Basis der Designentwürfe, und mit der von WordPress vorgesehenen Struktur in Verbindung mit HTML 5, CSS 3 und Javascript 6, wurde ein Theme entwickelt welches und sämtliche Grundstruktur von WordPress abdeckt. Um schnelle Updates der Hauptfunktionen bei Versions- Sprüngen von WordPress schnell und sauber abzuarbeiten, wurde für die individuellen Funktionen des GISA Systems ein Child Theme erstellt, welches das Parent Theme um die individuellen Funktionen erweitert.

Functions.php
In der Functions Datei werden sämtliche manuell angelegte Verhaltensweisen, Funktionen, Shortcodes, Visual Composer Erweiterungen sowie Widgets angelegt. Dies erfolgt stets über die von Wordpress und Visual Composer vorgegebene Logik, so ist ein sicheres Verhalten bei Updates von Wordpress gegeben. Des weiteren werden über die Functions Datei CSS und Javascript Dateien in das Theme eingebunden. Das Child Theme greift auf die Hauptfunktionen über die Functions Datei des Parent Themes zu und erweitert diese um eine verlustfreie Bearbeitung gewährleisten zu können. Auch Globale Funktionen wie die Slider und Referenzen, werden aus der Functions Datei in einen Shortcode in den Visual Composer geparst.
GISA Options
Die GISA Options sind die backend Einstellmöglichkeiten zum Theme, sie wurden zur einfachen Umstrukturierung des gesamten Themes angelegt. Sie dienen auch zur Visuellen Darstellung der im Main Theme verankerten Funktionen und Möglichkeiten. Hierbei wurde ein eigener Menüpunkt angelegt, aus dem das globale Verhalten & Aussehen heraus gesteuert und angepasst wird. Das Ziel war hier die Möglichkeit zu schaffen ohne Vorahnung von Programmierkenntnissen dem Nutzer die Möglichkeit zu geben am Theme mitzuarbeiten.
Shortcodes
Shortcode Erklärung
[referenzslider] Ruft über die Funktion referenz_slider und die WP_Query einen Card-Slider mit dem Inhalt der Referenzen auf.
[referenzgrid] Ruft über die Funktion referenzgrid und WP_Query ein Card-Grid mit dem Inhalt der Referenzen auf.
[neues-von-gisa] Ruft über die Funktion neues_von_gisa und WP_Query eine durchlaufende Liste der News Einträge auf.
[termin] Ruft über die Funktion termin und WP_Query (post_type, meta_key, meta_value, meta_compare) eine durchlaufende Liste der Events auf.
[contentBlocks postid=""] Ruft über die Funktion contentBlocks und der WP Funktion get_post den Inhalt einer Seite auf und gibt diesen aus. Weche Seite ausgegeben wird übermittelt man mit der postid, welche der auszugebenden Post/Page id entsprechen sollte.
[ziele kategorie=""] Ruft über die Funktion ziele_slider und WP_Query (post_type, post_per_page, cat) den Inhalt der Posts aus der gewählten Kategorie auf. Diese werden dann in einen Slider geladen und im Frontend dargestellt.

Slider & Carousel

GISA Slider
Der Gisa Slider hat diverse Funktion von Haus aus mitbekommen, die Hauptfunktion für das Projekt ist die Darstellung eines großen Hero Images in Verbindung mit einer Headline. Alternativ gibt es die Möglichkeit ein Video in den Hintergrund zu laden. Die Slider werden über eine eindeutige Kategorie und dessen Shortcode auf die gewünschten Seiten geladen. Der Shortcode kann über die Kategorien abgerufen werden und sieht wie folgt aus.
[qode_slider slider='kategorie_name' auto_start='true' animation_type='fade' slide_animation='6000' height='705' responsive_height='' background_color='' anchor='' show_navigation='yes' show_control='no'] Dieser Shortcode wird dann auf einer Seite im Meta Feld Select Slider Shortcode platziert und ist somit aktiv.
GISA Carousel
Über das Gisa Carousel können Logos von Partnerschaften, Zertifizierungen etc. eingepflegt und anschließend als Visual Composer Element auf Seiten eingebunden werden. Hierbei wird nur die Struktur des Karussells vorgegeben welche Inhalte letztendlich dargestellt werden liegt am Benutzer.

Custom Post Types

Custom Post Types
Die Custom Post Types beinhalten Inhaltselemente die nicht von Wordpress vorgegeben werden, aber der Wordpress Struktur und Funktion folgen. Für eine bessere Verwaltung wurden für diverse Inhalte z. B. „GISA Ziele„ Custom Post Types angelegt, und mit einer eigenen Inhaltsstruktur versehen. Dadurch können im Visual Composer diverse Elemente erzeugt werden, diese beziehen ihren Content aus den Post Types und so lässt sich Micro Content im Backend pflegen und im Frontend als z. B. Cards ausgegeben werden.

Einige Custom Post Types sind zum Beispiel:
  • GISA Slider
  • GISA Carousel
  • GISA Referenzen
  • Obsolet?? Neues aus den Medien
  • Obsolet?? Termine
  • Obsolet?? Aus den Social Networks
  • GISA Ziele
  • GISA Mitarbeiter

Formulare

Formulare
Als solide, stabile und bevorzugte Grundlage aller Formulare wurde ContactForm7 installiert. Über das Plugin werden diverse Funktionen zur Verfügung gestellt, mit "Hooks" und Erweiterungen für das Plugin gibt es hier nahezu unendlich Möglichkeiten. Die Hauptfunktion ist dabei klar definiert, die Fähigkeit schnell und einfach ein Kontaktformular mit der dazugehörigen E-Mail Template Struktur zu erstellen. Als Erweiterung wurde die Verbindung zwischen Formularen und Downloads installiert.

Die Formulare werden entweder über den Visual Composer eingebunden oder den vom Plugin generierten Shortcode. Beispiel Shortcode: [contact-form-7 id="21525" title="Kontaktformular 1"]

Downloads

Downloads
Über die Downloads können diverse Datei Typen zum Download angeboten werden, dabei ist auch die Möglichkeit geschaffen ein individuelles Icon festzulegen. In Verbindung mit den Formularen wurde eine Erweriterung hinzugefügt welche den Download erst nach absenden eines Formulars ermöglicht.

Die Einbindung erfolgt über ein neu erstelltes Visual Composer Element "Download". In diesem Element werden alle Downloads gesammelt und zur Auswahl gestellt. Der Ausgewählte Download wird dann mit der Struktur des Plugins dargestellt. Hier dient das Element nur als Schnittstelle zwischen Visual Composer und Download Plugin.

Mailpoet

Eigentlich ist Mailpoet ein komplettes Newsletter System. Für das Projekt wird allerdings nur die Anmeldefunktion benötigt, die gesammelten Daten lassen sich dann einfach im Dashboard exportieren. Das Plugin sammelte von Sich aus die IP-Adressen der Abonennten. Diese werden vom Plugin "Reset IP Adresse" aber mit dem Null Wert 127.0.0.1 überschrieben.

{SNAP} Social Network Auto-Poster

Das Plugin erlaubt es direkt aus Wordpress Beiträge zu diversen Social Media Kanälen zu posten. Dabei lässt sich über die Einstellungen die Ausgabe auf jedem Kanal individuell einstellen. Man legt also ein Template für jeden Kanal an, wie sich das Plugin bei veröffentlichung verhalten soll.

In Plugin selbst lassen sich die Postings zu verschiedenen Zeiten ausgeben, dabei wird zwischen dem direkten Posten und das Posten über einen Pseudo Cron Job von Wordpress unterschieden. Auch lassen sich Globale Einstellungen vornehmen wie ein URL Shortener, HTTPS Force, Auto Hashtags, und die Zeichenlänge.

SEO & Performance

YOAST SEO
YOAST SEO, ist die All-in-one Lösung im SEO Bereich. Nahezu alle SEO relevanten Einstellungen und Funktionen können hier festgelegt werden.
W3 Total Cache
Das effizienteste Plugin für die Verwaltung der Cache Funktionen. Dabei ist bei dem Einstellen immer ein Auge auf das Front End zu werfen, hier können kleine Änderungen das Front End unerreichbar machen. Mit den richtigen Einstellungen kann hier ein massiver Performance Boost erreicht werden.
Reset IP Adresse
Ein Helfer Plugin von nviimedia für die anonymisierung der IP Adresse der Nutzer.
WP Better Email
Stellt Emails aus Wordpress mit dem HTML Content Type dar.
Duplicate Post
Ermöglicht das duplizieren von vorhanden Beiträgen und Seiten.
